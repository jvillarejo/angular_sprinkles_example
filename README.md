# Angular Sprinkles Example Application

This repository is an example usage of the [`angular_sprinkles`](https://github.com/BrewhouseTeam/angular_sprinkles)
gem in a Rails Application. It uses the [TODO MVC](https://github.com/tastejs/todomvc) template and makes heavy use
of the gem's features.

A live demo of this example can be found [here](http://sprinkles-example.herokuapp.com/).
